
# Install gcloud

1. Enter the following at a command prompt:

curl https://sdk.cloud.google.com | bash

2. Restart your shell:

exec -l $SHELL

3. Run gcloud init to initialize the gcloud environment:

gcloud init

Finish all the questions asked through prompt.
